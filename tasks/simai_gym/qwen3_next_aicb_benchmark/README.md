# Qwen3-Next AICB Benchmark

This is the default real Vidur+AICB benchmark profile for the local CUDA host.
Unlike the smoke task, it runs a multi-request synthetic workload and is intended
for comparing scheduler and cluster configuration changes.

Prerequisites:

- `.venv311` with Vidur and AICB dependencies installed.
- CUDA toolkit at `/usr/local/cuda-12.9`.
- `scripts/deepgemm_nvcc_sm89_wrapper.sh` for RTX 5880/Ada hosts.
- AICB cache/CSV generated by the runner or by `scripts/run_real_simai_vidur_aicb_benchmark.sh`.

Run:

```bash
python -m mlsysbench.simai_bench evaluate \
  --task tasks/simai_gym/qwen3_next_aicb_benchmark \
  --submission submissions/examples/qwen3_next_aicb_benchmark_baseline.json
```

The runner fails the evaluation if Vidur reports AICB default/fallback messages
such as `AICB data is empty` or `Expected CSV file was NOT created`.
