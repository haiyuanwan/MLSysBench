# Qwen3-Next AICB Smoke Task

This task exercises the real Vidur runner with AICB-backed layer timings. It is
intended as a CUDA-host smoke task, not a full benchmark sweep.

Prerequisites:

- `.venv311` with Vidur and AICB dependencies installed.
- CUDA toolkit at `/usr/local/cuda-12.9`.
- `scripts/deepgemm_nvcc_sm89_wrapper.sh` for RTX 5880/Ada hosts.
- Real AICB CSV/cache generated by `scripts/run_real_simai_vidur_aicb_smoke.sh`
  or generated on demand by the runner.

Run:

```bash
python -m mlsysbench.simai_bench evaluate \
  --task tasks/simai_gym/qwen3_next_aicb_smoke \
  --submission submissions/examples/qwen3_next_aicb_smoke_baseline.json
```

The runner fails the evaluation if Vidur reports AICB default/fallback messages
such as `AICB data is empty` or `Expected CSV file was NOT created`.
